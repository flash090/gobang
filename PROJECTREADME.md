# Gobang Online - 五子棋联机版

一个基于 Next.js + Supabase + Vercel 的多人五子棋网页游戏项目。

## 🧩 项目目标
- 用户可以通过链接访问页面，在浏览器中双人或多人对战。
- 不需要安装，手机和电脑都能玩。
- 尽量保持界面美观、操作简单。

## 🛠️ 技术栈
- Next.js (React 框架)
- Supabase (数据库 & 实时通信)
- Vercel (托管平台)
- Tailwind CSS (界面美化)
- TypeScript
